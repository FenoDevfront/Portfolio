# Portfolio - RATSIMBAHARISOA Fenoniaina

Portfolio personnel de **Fenoniaina RATSIMBAHARISOA**, Chef de projet & Développeur Web Full-Stack basé à Antananarivo, Madagascar.

## Aperçu

Ce portfolio présente mon parcours professionnel, mes compétences techniques et mes projets réalisés. Il est construit avec React et Vite pour des performances optimales.

## Technologies utilisées

### Frontend
- **React 18** - Framework JavaScript
- **Vite** - Build tool rapide
- **Framer Motion** - Animations fluides
- **React Icons** - Icônes
- **Axios** - Requêtes HTTP

### Design
- CSS3 avec effets glassmorphism
- Responsive design (mobile-first)
- Animations et transitions modernes

## Fonctionnalités

- Design moderne avec effets de verre (glassmorphism)
- Animations fluides au scroll
- Sections : Hero, À propos, Expériences, Projets, Compétences, Contact
- Entièrement responsive
- Mode sombre par défaut
